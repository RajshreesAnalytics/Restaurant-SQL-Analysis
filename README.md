# 🍽️ Restaurant Sales Analysis (SQL Project)

## 📌 Project Overview
This project analyzes restaurant sales by combining menu data with order transactions to extract business insights.

---

## 🎯 Business Questions Answered
- Total number of orders
- Total revenue generated
- Top 5 most ordered items
- Least ordered items
- Orders by category
- Revenue by category

---

## 📂 Dataset
- menu_items.csv  
- order_details.csv  

---

## 🛠️ Tools Used
- MySQL  
- SQL  
- GitHub  

---

## 💡 Key Insights
- A few items drive most of the sales  
- Some categories perform better than others  
- High price does not always mean high demand
